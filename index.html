Etiquetas em, strong, code, q y blockquote

La etiqueta em se utiliza para dar énfasis a una palabra. Tiene un uso similar a la etiqueta strong que usaremos cuando queramos marcar una palabra como importante: Por ejemplo:

el perro es un animal muy fiel considerado en muchos casos el mejor amigo del hombre encontrar la siguiente referencia a la etiqueta strong en www.campusmvp.es :

La etiqueta strong realmente no está diseñada para dar negrita, aunque el resultado sea en realidad ese. Esta etiqueta es semántica, y por lo tanto sirve para dar contexto al contenido de la página. En este caso la etiqueta sirve para indicar que lo que va dentro de ella reviste importancia o urgencia. Para marcar palabras o fragmentos de texto que hacen referencia a código de programación usaremos la etiqueta code La etiqueta q la utilizaremos para marcar citas cortas.
